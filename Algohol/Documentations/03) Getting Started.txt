Algohol (Version: 1.0.0)
------------------------

GETTING STARTED
---------------

Basic information:-

-> This library uses the following header files (already included):-
	-> stdio.h
	-> iostream
	-> cstdlib
-> Also it includes "std" namespace from "iostream".
-> It heavily uses pointers and low level memory management functions (from "cstdlib").
-> Import the library to your project by writing (in your code file):
#include "algohol.hpp"


About namespaces:-

-> There are total 3 namespaces in the library: data_structures, searching_algorithms & sorting_algorithms.
-> The namespace "data_structures" contains classes named: BST, LinkedList, Stack & Queue.
-> The namespace "searching_algorithms" contains functions named: linSearch & binSearch.
-> The namespace "sorting_algorithms" contains functions named: bubble, bucket, insertion, merge, quick, radix, selection & shell.
